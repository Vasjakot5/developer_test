Инструкция по запуску проекта
1. Войдите в папку config файл db.php.
   
1.1. Измените имя пользователя и пароль или оставьте так + по желанию можете изменить название базы данных.

2. Откройте PhpMyAdmin или другое веб-приложение для работы с базами данных и создайте там свою БД.
2.1 Затем импортируйте в свою БД файл developer_test.sql, который находится в корне проекта.

3. Закиньте проект на хостинг(или используйте локальный сервер по типу: OpenServer или XAMPP) и запустите его.

4. Проверьте, чтобы всё было подключенно и введены правильные данные
